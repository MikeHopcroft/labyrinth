# Labyrinth CLI

Most `labyrinth` functionality is available in the `analyze.js` command-line tool. You can run this tool with the `-h` flag to get help:

[//]: # (spawn node build\src\apps\analyze.js -h)
~~~
$ node build\src\apps\analyze.js -h

~~~
